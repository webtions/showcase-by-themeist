This plugin creates a showcase CPT which can be used to create a website to showcase website designs

== Description ==

This plugin creates a showcase custom post type with meta boxes.

= Features of Showcase plugin =

- Creates showcase CPT
- Integrates with "I Recommend This plugin"
- Metabox to add URL of external website.

== Installation ==

1. Upload the directory `/showcase-by-themeist/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create custom template inside your theme to display the showcase
